# 🍽 Modern Restaurant Landing Page

## 📌 Description
This project is a modern and responsive restaurant landing page built using HTML, CSS, JavaScript, and jQuery. It showcases a clean UI design along with interactive features like smooth navigation and menu filtering.

---

## 🚀 Features
- Responsive landing page design
- Fixed navigation bar
- Smooth scrolling between sections
- Interactive menu filtering (Veg / Non-Veg / Drinks)
- Hover effects and modern UI styling

---

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- jQuery

---

## ⭐ Unique Feature
The project includes an interactive menu filtering system using jQuery, allowing users to dynamically view food categories without reloading the page.

---

## 📖 How to Use
1. Open `index.html` in browser
2. Use navbar to navigate sections
3. Use menu filters to explore items
